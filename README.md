## Répartition des "Autre Fonctionnalité"

Les autres fonctionnalités demandées sont réparties dans les pages comme suit:

* Page Home
  * Vibration
  * TextToSpeech
* Page Scanner
  * Toast
* Page Video
  * Toast
  * Shake
  * BarcodeScanner
  * SocialSharing
* Page Gmap
  * GoogleMaps
  * Gyroscope

### Installation:

Pour se rappeler des commandes:

```bash
$ npm install
$ npm cordova platform add android
$ npm cordova run android
```
